# jboogxcreative

> A modern portfolio landing page for a videographer, storyteller, and content creator.

## Features

- **TypeScript** - Type-safe development
- **Vite** - Lightning-fast build tooling
- **Responsive Design** - Mobile-first approach
- **Typing Animation** - Dynamic name reveal effect
- **Smooth Animations** - Fade-in transitions and parallax effects

## Quick Start

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Project Structure

```
jboogxcreative-landing-page/
├── public/          # Static assets
├── src/
│   ├── main.ts      # TypeScript logic
│   └── style.css    # Styles
├── index.html       # Entry point
└── package.json
```

## Tech Stack

- TypeScript
- Vite
- CSS3 with Animations
- Modern ES Modules

## Development

The development server runs at `http://localhost:3000` with hot module replacement enabled. Any changes to source files will automatically refresh the browser.
